# Unicode Enigma

A Unicode-based Enigma machine implemented in Python, supporting the full Unicode scalar range
(excluding surrogate code points), with a graphical user interface.

## Features
- Enigma-style rotor encryption
- Full Unicode support (emoji, CJK, symbols, etc.)
- Plugboard, ring settings, notches, and double-stepping logic
- Deterministic encryption via passphrase
- Built-in GUI (Tkinter)

## Requirements
- Python 3.8+
- Tkinter (usually included with Python)

## How to Run
```bash
python unicode_enigma.py
